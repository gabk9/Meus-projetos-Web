===== Abreviações Emmet no HTML =====


div*1 → Cria uma <div></div>. (Exemplo básico de tag).
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
.classe → Cria uma <div> com classe. (Exemplo: .caixa gera <div class="caixa"></div>).
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
#id → Cria uma <div> com ID. (Exemplo: #menu gera <div id="menu"></div>).
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
ul>li*3 → Cria uma lista não ordenada com 3 itens <li>. (Exemplo: <ul><li></li><li></li><li></li></ul>).
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
nav>ul>li*3>a → Cria uma estrutura de navegação com 3 links. (Exemplo: <nav><ul><li><a></a></li>...</ul></nav>).
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
table>tr*2>td*3 → Cria uma tabela com 2 linhas e 3 colunas. (Exemplo: <table><tr><td></td><td></td><td></td></tr>...</table>).
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
header+main+footer → Cria as 3 seções principais do HTML5 uma após a outra.
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
.menu>.item*4 → Cria uma div com classe "menu" contendo 4 divs com classe "item".
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
form:post>input:text+input:submit → Cria um formulário com método POST, um campo de texto e um botão submit.
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
input:checkbox*3 → Cria 3 checkboxes. (Exemplo: <input type="checkbox">...).
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
h1{Título da Página} → Cria um h1 com o texto "Título da Página". (Exemplo: <h1>Título da Página</h1>).
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
p{Texto aqui}*3 → Cria 3 parágrafos com o texto "Texto aqui".
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
a[href="#"]{Clique aqui} → Cria um link com href e texto "Clique aqui". (Exemplo: <a href="#">Clique aqui</a>).
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
img[src="imagem.jpg" alt="Descrição"] → Cria uma imagem com src e alt. (Exemplo: <img src="imagem.jpg" alt="Descrição">).
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
button.btn{Clique} → Cria um botão com classe "btn" e o texto "Clique". (Exemplo: <button class="btn">Clique</button>).
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------



===== Símbolos das Abreviações Emmet (HTML) =====



. → Classe (class)
Exemplo: .caixa → <div class="caixa"></div>
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
# → ID (id)
Exemplo: #menu → <div id="menu"></div>
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
> → Filho direto (hierarquia)
Exemplo: ul>li → <ul><li></li></ul>
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
+ → Elemento irmão (no mesmo nível)
Exemplo: header+main+footer → <header></header><main></main><footer></footer>
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
* → Repetição (quantidade)
Exemplo: li*3 → <li></li><li></li><li></li>
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
{} → Conteúdo de texto
Exemplo: h1{Título} → <h1>Título</h1>
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
[] → Atributos personalizados
Exemplo: a[href="#"] → <a href="#"></a>
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
: → Tipo de elemento especial (Exemplo: inputs)
Exemplo: input:checkbox → <input type="checkbox">
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
$ → Contador incremental
Exemplo: item$*3 → <div class="item1"></div><div class="item2"></div><div class="item3"></div>
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
() → Agrupamento
Exemplo: (header>ul>li*2)+footer → Cria o header com lista e depois o footer
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
^ → Subir um nível (voltar para o pai)
Exemplo: div>p>span^h1 → <div><p><span></span></p><h1></h1></div>
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
