Operador             Nome                                      Descrição


==                   Igualdade frouxa                          Compara valores com conversão de tipo
------------------------------------------------------------------------------------------------------------------------
===                  Igualdade estrita                         Compara valores e tipos, sem conversão
------------------------------------------------------------------------------------------------------------------------
!=                   Diferença frouxa                          Verifica se os valores são diferentes (com conversão)
------------------------------------------------------------------------------------------------------------------------
!==                  Diferença estrita                         Verifica se os valores ou tipos são diferentes
------------------------------------------------------------------------------------------------------------------------
>                    Maior que                                Verifica se o valor da esquerda é maior
------------------------------------------------------------------------------------------------------------------------
<                    Menor que                                Verifica se o valor da esquerda é menor
------------------------------------------------------------------------------------------------------------------------
>=                   Maior ou igual                           Verifica se é maior ou igual
------------------------------------------------------------------------------------------------------------------------
<=                   Menor ou igual                           Verifica se é menor ou igual
------------------------------------------------------------------------------------------------------------------------
