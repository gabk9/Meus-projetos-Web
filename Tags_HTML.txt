
===== Tags usadas no html=====


<!DOCTYPE> → Define o tipo de documento como HTML5.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<html> → Elemento raiz de toda a página HTML.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<span> → Define um trecho específico de texto inline, usado geralmente para aplicar estilos ou manipular via JavaScript sem quebrar o fluxo do texto.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<head> → Contém metadados da página (informações que não aparecem no conteúdo visual).
----------------------------------------------------------------------------------------------------------------------------------------------------------
<meta> → Define metadados, como charset, descrição, autor, etc.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<title> → Define o título que aparece na aba do navegador.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<bdo> → Controla a direção do texto (ex: da direita para a esquerda, como em árabe).
----------------------------------------------------------------------------------------------------------------------------------------------------------
<link> → Faz ligação com arquivos externos (ex: CSS).
----------------------------------------------------------------------------------------------------------------------------------------------------------
<style> → Permite adicionar CSS direto dentro do HTML.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<body> → Contém todo o conteúdo visível da página.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<header> → Define o cabeçalho da página ou seção.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<h1> até <h2> → Títulos e subtítulos da página. 
----------------------------------------------------------------------------------------------------------------------------------------------------------
<p> → Parágrafo.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<mark> → Marca um texto com fundo destacado.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<hr> → Insere uma linha horizontal.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<br> → Quebra de linha.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<b> → Deixa o texto em negrito.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<strong> → Dá importância semântica ao texto (também deixa negrito).
----------------------------------------------------------------------------------------------------------------------------------------------------------
<i> → Texto em itálico.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<u> → Sublinha o texto.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<small> → Texto pequeno.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<del> → Texto riscado.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<img> → Exibe uma imagem.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<sup> → Texto sobrescrito (em cima, como expoente).
----------------------------------------------------------------------------------------------------------------------------------------------------------
<sub> → Texto subscrito (em baixo, como CO₂).
----------------------------------------------------------------------------------------------------------------------------------------------------------
<table> → Cria uma tabela.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<tr> → Linha da tabela.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<th> → Cabeçalho de uma coluna na tabela.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<td> → Célula de dado da tabela.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<ul> → Lista não ordenada.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<ol> → Lista ordenada.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<li> → Item de lista.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<a> → Link (âncora).
----------------------------------------------------------------------------------------------------------------------------------------------------------
<iframe> → Quadro interno para mostrar outra página.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<form> → Formulário.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<label> → Rótulo de campo no formulário.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<input> → Campo de entrada.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<select> → Menu suspenso.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<option> → Opções de um select.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<textarea> → Área de texto de múltiplas linhas.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<section> → Seção da página (tag semântica).
----------------------------------------------------------------------------------------------------------------------------------------------------------
<nav> → Área de navegação (tag semântica).
----------------------------------------------------------------------------------------------------------------------------------------------------------
<div> → Container genérico (divisória).
----------------------------------------------------------------------------------------------------------------------------------------------------------
<article> → Conteúdo independente.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<footer> → Rodapé.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<video> → Player de vídeo.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<source> → Fonte de mídia (usado dentro de <video> e <audio>).
----------------------------------------------------------------------------------------------------------------------------------------------------------
<audio> → Player de áudio.
----------------------------------------------------------------------------------------------------------------------------------------------------------


===== Outras Tags HTML5 Que Existem =====


<h3> até <h6> → Títulos adicionais.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<main> → Conteúdo principal da página.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<aside> → Conteúdo lateral (ex: barras laterais).
----------------------------------------------------------------------------------------------------------------------------------------------------------
<figure> → Container para imagens, gráficos ou outros conteúdos.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<figcaption> → Legenda de uma figura.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<time> → Representação de uma data ou hora.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<abbr> → Abreviação.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<address> → Informações de contato.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<cite> → Citação de uma fonte.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<code> → Exibe código.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<pre> → Texto pré-formatado.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<blockquote> → Bloco de citação.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<details> → Seção expansível.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<summary> → Resumo/legenda de um <details>.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<progress> → Barra de progresso.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<meter> → Medida em escala.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<canvas> → Área gráfica para desenhar via JavaScript.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<svg> → Gráficos vetoriais.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<embed> → Incorpora conteúdo externo.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<object> → Incorpora objetos multimídia.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<param> → Parâmetros de um <object>.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<track> → Legenda para vídeo e áudio.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<template> → Fragmento de conteúdo que pode ser clonado por JavaScript.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<noscript> → Conteúdo para quando o JavaScript está desabilitado.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<output> → Resultado de cálculo de formulários.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<fieldset> → Agrupa campos de formulário.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<legend> → Título de um fieldset.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<datalist> → Lista de sugestões para inputs.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<optgroup> → Agrupamento de opções dentro de um select.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<canvas> → Área de desenho (para gráficos, jogos, etc).
----------------------------------------------------------------------------------------------------------------------------------------------------------
<dialog> → Caixas de diálogo/modal.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<menu> → Agrupamento de comandos.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<menuitem> → Item de um menu.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<slot> → Usado em Web Components.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<del> → Texto deletado (já usado por você).
----------------------------------------------------------------------------------------------------------------------------------------------------------
<ins> → Texto inserido.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<map> → Mapeamento de imagem.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<area> → Área clicável dentro de um map.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<caption> → Legenda da tabela.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<colgroup> → Grupo de colunas na tabela.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<col> → Define estilo de colunas.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<thead>, <tbody>, <tfoot> → Agrupamentos de partes da tabela.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<base> → Define a URL base de todos os links relativos da página.
----------------------------------------------------------------------------------------------------------------------------------------------------------
<script> → Código JavaScript dentro do HTML.
----------------------------------------------------------------------------------------------------------------------------------------------------------
